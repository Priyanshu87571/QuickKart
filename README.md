# 🛒 QuickKart – Quick Commerce Frontend

QuickKart is a modern, responsive quick-commerce web application frontend built using **Next.js**, **TypeScript**, and **Tailwind CSS**.  
It simulates a real-world grocery delivery platform with a complete customer and admin experience.

## 🚀 Project Overview

QuickKart provides a fast and smooth shopping experience for users and a management interface for admins.  
The frontend is designed to be scalable, responsive, and ready for backend integration.

---

## ✨ Key Features

### 👤 Customer Features
- Browse products by category
- Product listing with price, rating, and discounts
- Product detail pages with images and descriptions
- Add to cart and manage quantities
- Persistent cart using localStorage
- Toast notifications for user actions
- Checkout flow with address and payment UI
- Order history page
- User profile page with editable information
- Saved addresses UI
- Fully responsive UI (mobile, tablet, desktop)

---

### 🛠 Admin Features (UI)
- Admin dashboard overview
- Product management UI
- Order management UI
- Protected admin routes (UI-level protection)

---

## 🧰 Tech Stack

- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **State Management:** Zustand
- **Notifications:** React Hot Toast
- **Icons:** Lucide React

## 🧪 Getting Started

### 1. Clone the Repository
git clone https://github.com/your-username/QuickKart.git

### 2. Navigate to Frontend
cd frontend

### 3. Install Dependencies
npm install

### 4. Run Development Server
npm run dev

Application will run on:
http://localhost:3000
